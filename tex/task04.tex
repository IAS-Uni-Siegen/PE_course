\task{Transistor-based AC/DC converter (single-phase AFE)}

\taskGerman{Transistorbasierter AC/DC-Umrichter (einphasiger AFE)}

An electric-drive test bench is connected to a single-phase grid via a full-bridge transistor AFE rectifier.
The converter should operate with approximately unity power factor and a stable DC-link voltage.
During testing, increased DC-link ripple is observed and the PWM strategy is questioned.

\begin{germanblock}
Ein elektrischer Antriebsprüfstand ist über einen einphasigen Vollbrücken-Transistor-AFE-Gleichrichter
an das Netz angeschlossen. Der Umrichter soll mit näherungsweise Leistungsfaktor 1 und stabiler
DC-Zwischenkreisspannung betrieben werden. Während der Tests wird eine erhöhte Zwischenkreiswelligkeit
beobachtet und die PWM-Strategie wird hinterfragt.
\end{germanblock}

\subtask{For approximately unity power factor operation, determine the required grid current RMS value and
the corresponding grid current peak value.}{0}
\subtaskGerman{Bestimmen Sie für den Betrieb mit näherungsweise Leistungsfaktor 1 den erforderlichen
Effektivwert des Netzstroms sowie den zugehörigen Spitzenwert des Netzstroms.}

\subtask{Calculate the required converter input-voltage amplitude $\hat{u}_2$ from the phasor relation and verify
whether the DC-link voltage is sufficient for boost-type rectifier operation.}{0}
\subtaskGerman{Berechnen Sie aus der Zeigerbeziehung die erforderliche Amplitude der Umrichtereingangsspannung $\hat{u}_2$
und prüfen Sie, ob die DC-Zwischenkreisspannung für den Hochsetzbetrieb ausreicht.}

\subtask{Classify the operating mode for each case below and justify using power-flow direction:}{0}
\begin{itemize}
    \item Case A: $P=+\SI{8}{\kilo\watt}$,
    \item Case B: $P=-\SI{5}{\kilo\watt}$.
\end{itemize}
\subtaskGerman{Ordnen Sie für die folgenden Fälle die Betriebsart zu und begründen Sie mit der Leistungsflussrichtung:
\begin{itemize}
    \item Fall A: $P=+\SI{8}{\kilo\watt}$,
    \item Fall B: $P=-\SI{5}{\kilo\watt}$.
\end{itemize}}

\subtask{Using the steady-state ripple model, estimate the DC-link voltage oscillation amplitude
$\hat{u}_C$ and the peak-to-peak ripple $\Delta U_{dc,pp}\approx2\hat{u}_C$.}{0}
\subtaskGerman{Schätzen Sie mit dem stationären Welligkeitsmodell die Amplitude der Zwischenkreisspannungsschwingung
$\hat{u}_C$ und die Spitze-Spitze-Welligkeit $\Delta U_{dc,pp}\approx2\hat{u}_C$.}

\subtask{For $s^*=0.5$, compare complementary PWM and interleaved PWM regarding current-ripple amplitude.
Quantify the ratio and state which strategy is preferable.}{0}
\subtaskGerman{Vergleichen Sie für $s^*=0.5$ komplementäre PWM und interleavte PWM hinsichtlich der Stromwelligkeitsamplitude.
Quantifizieren Sie das Verhältnis und geben Sie an, welche Strategie vorzuziehen ist.}
